# Laboratory Inventory Dashboard
An Excel-based laboratory inventory management dashboard for analyzing inventory value, stock movement, inventory status, suppliers, and storage locations.

## Project Overview

This project is an inventory management dashboard developed using Microsoft Excel.

The project analyzes laboratory inventory data to provide insights into inventory value, stock movement, inventory status, suppliers, storage locations, and high-value items.

## Objectives

This project aims to:

- Monitor total inventory value
- Analyze inventory value by category, supplier, and storage location
- Compare incoming and outgoing inventory
- Identify inventory items with low stock or no stock
- Identify the top 5 items with the highest inventory value
- Create an inventory search system based on item codes

## Tools & Excel Skills

- Microsoft Excel 2019
- PivotTable
- PivotChart
- Slicer
- INDEX-MATCH
- SUMIFS
- COUNTIFS
- Data Cleaning
- Dashboard Design

## Dashboard

The dashboard provides an interactive overview of laboratory inventory through KPI cards, PivotCharts, and Slicers.

### Key Performance Indicators

- Total Inventory Value
- Total Suppliers
- Total Low-Stock Items
- Total Incoming Inventory
- Total Outgoing Inventory

### Analysis Included

- Inventory Value by Category
- Inventory Value by Supplier
- Inventory Value by Storage Location
- Inventory Status
- Top 5 Items by Inventory Value
- Incoming vs Outgoing Inventory

## Workbook Structure

| Sheet | Description |
|---|---|
| Raw Data | Original laboratory inventory dataset |
| Data Olahan | Processed and calculated inventory data |
| PIVOT | PivotTables and PivotCharts used for analysis |
| Dashboard | Interactive inventory dashboard |
| Search Inventory | Inventory search tool using INDEX-MATCH |

## Inventory Search Tool

The Search Inventory sheet allows users to search for inventory information by entering an item code.

The search tool retrieves information such as:

- Item Name
- Category
- Unit
- Supplier
- Stock
- Inventory Status
- Other inventory information

This feature was created using the `INDEX-MATCH` function.

## Project Outcome

This project demonstrates the use of Microsoft Excel for inventory data analysis, reporting, and dashboard development.

The dashboard transforms raw inventory data into a structured and interactive report that can support inventory monitoring and decision-making.
